# Mundo Mascotas Frontend

Aplicación web frontend para la gestión de mascotas y citas veterinarias.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript

## Estructura del proyecto
- `assets/`: imágenes e íconos
- `pages/`: páginas internas
- `styles/`: hojas de estilo
- `scripts/`: lógica JavaScript
- `docs/`: documentación de apoyo

## Módulos iniciales
- Dashboard principal
- Gestión de mascotas
- Gestión de citas

## Ejecución del proyecto
Abrir el archivo `index.html` en el navegador.
